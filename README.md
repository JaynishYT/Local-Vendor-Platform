# Final Using React

Local vendor booking platform with a React client and an Express/MongoDB server.

## Setup

Install dependencies:

```bash
npm install
```

Create a `.env` file from `.env.example`, then start the app:

```bash
npm run dev
```

The server uses `PORT`, `MONGODB_URI`, `ADMIN_USERNAME`, `ADMIN_PASSWORD`, and `ADMIN_SECRET` from the environment.
